# GeckOS 灵蜥面板

轻量级 Linux 主机管理系统，开箱即用，通过浏览器即可完成服务器的全部日常运维。

## 功能模块

### 🖥️ Web 终端
完整的浏览器终端，随时随地通过网页操作服务器，支持多会话、中文输入、复制粘贴，无需 SSH 客户端。

### 📁 文件管理
可视化的文件管理器，支持浏览、上传（断点续传）、下载、编辑、解压、批量操作，多磁盘一目了然。

### 🐳 Docker 管理
容器、镜像、卷的完整管理，支持启停、日志查看、端口映射、 compose 部署，Docker 运维不再依赖命令行。

### 📊 系统监控
CPU、内存、磁盘、网络的实时监控与历史趋势，掌握主机运行状态，及时发现问题。

### ⚙️ 进程管理
查看与管理系统进程，了解资源占用情况，支持结束异常进程。

### 📝 系统日志
实时日志流推送，无需 `tail -f`，在浏览器即可追踪系统动态。

### 🏪 应用商店
一键部署 Docker 应用 — Nginx、MySQL、Redis、WordPress 等，按分类浏览、配置环境变量、实时查看安装进度。

### 🔌 插件系统
丰富的插件生态扩展面板能力，从插件市场安装插件，按需扩展功能。

### 📱 多端支持
Web 端 + 移动端（iOS / Android），手机也能轻松管理服务器。

## 特色

- **零配置**：安装即用，无需复杂部署
- **轻量级**：单文件二进制，资源占用极低，适合各种规格的服务器与嵌入式设备
- **安全可靠**：JWT 认证、默认密码强制修改、配置自动生成
- **现代化界面**：响应式设计，桌面/移动端自适应
- **插件化架构**：按需安装功能模块，保持核心精简

## 快速开始

```bash
# 一键安装
curl -fsSL https://www.geckosweb.cn/install.sh | bash

# 启动服务
geckos serve
```

安装完成后访问终端输出的地址，默认账号 `admin` / `123456`（首次登录请立即修改）。
