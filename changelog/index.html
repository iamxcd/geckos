更新日志

v0.8.0

  • 新增手机端 App（iOS / Android），支持局域网自动发现、移动端终端、文件传输
  • 新增 AI 应用一键部署，支持 OpenClaw 龙虾AI、Hermes 爱马仕AI
  • 优化 Docker 容器管理界面，新增批量操作
  • 修复文件管理在大文件上传时的进度显示问题

v0.7.0

  • 新增插件系统，支持一键安装 AdGuard Home、Alist、PhotoPrism 等应用
  • 新增系统监控页面，实时显示 CPU、内存、磁盘、网络数据
  • 优化 Web 终端，支持多标签页和快捷键
  • 修复进程管理在特殊字符进程名下的显示异常

v0.6.0

  • 新增文件管理模块，支持拖拽上传、在线编辑、压缩解压
  • 新增 Docker 可视化管理，容器启停、日志查看、镜像管理
  • 优化安装脚本，支持自动检测端口冲突
  • 修复 systemd 服务在特定发行版下的兼容性问题

v0.5.0

  • 首个公开版本
  • Web 终端，浏览器直接执行 Linux 命令
  • 基础文件浏览功能
  • 系统信息概览
