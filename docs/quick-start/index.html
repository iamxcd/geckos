安装 GeckOS

系统要求

说真的，GeckOS 对硬件要求低得离谱——毕竟纯 Go 开发，单二进制文件零依赖，你家里吃灰的那个小主机、树莓派，甚至那个 512M 内存的 VPS 都能跑起来：

配置最低要求推荐配置
CPUx86_64 / ARM64任意现代处理器
内存512 MB1 GB+
磁盘100 MB1 GB+
系统LinuxUbuntu / Debian / CentOS
权限rootroot

GeckOS 需要 root 权限运行，毕竟要管 Docker、管文件、管系统，没点权限怎么干活。

一键安装（强烈推荐）

复制这一行，粘贴到终端，回车，完事：

1

curl -fsSL https://www.geckosweb.cn/install.sh | sudo bash

脚本会帮你把脏活累活全干了：

  1. 检查系统环境 — 确认操作系统、CPU 架构，检查必要命令（wget、unzip）是否就位
  2. 随机分配端口 — 在 10000-65535 范围里找个没被占用的端口，省得跟你别的服务打架
  3. 获取最新版本 — 从云端拉最新版本号，永远不落伍
  4. 下载并安装 — 安装包下载到 /opt/geckos/
  5. 配置环境变量 — 创建符号链接到 /usr/local/bin/geckos，在哪都能直接敲 geckos
  6. 设置数据目录 — 创建 /geckosdata，文件管理就靠它
  7. 配置服务 — 创建 systemd 服务并设置开机自启，重启也不怕
  8. 启动服务 — 一切就绪，直接启动

安装完注意看

装完终端会打印访问地址和默认账号，别手滑关了窗口：

  • 默认账号：admin / 123456
  • 配置文件：/opt/geckos/config.yaml
  • 数据目录：/opt/geckos
  • 文件管理数据：/geckosdata

⚠️ 首次登录后第一件事——改密码！别问我是怎么知道 123456 不安全的。

5 分钟上手

1. 文件管理

左侧菜单点「文件管理」，像操作电脑一样拖拽上传、在线编辑文件、解压压缩包，不用敲一行命令。

2. Docker 容器

点「Docker」，可以看到所有容器的运行状态，一键启停、查看日志，比命令行直观多了。

3. Web 终端

还是想敲命令？点「终端」，浏览器里直接连上服务器 shell，自带常用快捷键。

4. 系统监控

「监控」页面实时显示 CPU、内存、磁盘、网络数据，服务器状态一目了然。

5. 安装插件

点「插件」→「云端」，浏览可用的插件扩展，一键安装到本地。

6. 应用商店

打开桌面「应用商店」图标，浏览 Nginx、MySQL、Redis、WordPress 等 Docker 应用，填写环境变量一键部署，安装进度实时可见。

7. 手机端管理

装个手机 App，躺在床上也能管服务器。打开 App 自动扫描局域网发现 GeckOS 主机，IP 变了也不怕。手机上能敲终端、传文件、管 Docker、看监控，出门紧急排错不用找电脑。

详细功能见下方 手机端 章节。

手机端

躺在床上也能管服务器，GeckOS 有原生移动端，iOS 和 Android 都能装：

为什么装手机端？

家里服务器 IP 经常变？路由器重启就换 IP？装个手机端就完事了——App 会自动扫描局域网发现 GeckOS 主机，不用记 IP，不用手动输地址，打开 App 直接连。

核心功能

  • 局域网自动发现 — UDP 广播扫描，自动找到局域网内的 GeckOS 主机，IP 变了也不怕
  • 移动端终端 — 手机上也能敲命令，自带虚拟键盘，出门紧急排错不用找电脑
  • 文件传输 — 手机和服务器互传文件，断点续传不怕中断
  • Docker 管理 — 手机上查看容器状态、日志，随时启停容器
  • 系统监控 — CPU、内存、磁盘、网络实时数据，躺沙发上看也行
  • 文件管理 — 浏览、编辑、上传下载，手机上也能操作服务器文件
  • 插件管理 — 安装、更新插件，手机上一键搞定

连接服务器

  1. 自动发现（推荐） — 手机和服务器连同一 WiFi，打开 App 自动扫描出局域网内的 GeckOS 主机，点一下就能连，再也不怕 IP 变了
  2. 手动连接 — 不在同一局域网时，手动输入服务器地址和端口（建议配合 DDNS 或内网穿透使用）
  3. 用 Web 端的账号密码登录即可

💡 提示：如果服务器 IP 经常变，推荐搭配 GeckOS 的 DDNS 插件使用，手机端通过域名连接，彻底告别 IP 烦恼。

卸载

交互式卸载（推荐）

会问你备不备份数据，比较稳妥：

1

curl -fsSL https://www.geckosweb.cn/uninstall.sh | sudo bash

或者先下载再执行，心里更踏实：

1
2

wget https://www.geckosweb.cn/uninstall.sh
sudo bash uninstall.sh

卸载流程：

  1. 停止服务 — 先把 GeckOS 停下来
  2. 禁用服务 — 移除 systemd 服务文件
  3. 备份数据 — 问你要不要备份（非交互模式直接跳过）
  4. 删除安装目录 — /opt/geckos/ 整个移除
  5. 清理环境变量 — 删掉 /usr/local/bin/geckos 符号链接，干干净净

非交互式卸载

管道方式执行时，脚本默认不备份直接卸载，适合脚本化操作：

1

curl -fsSL https://www.geckosweb.cn/uninstall.sh | sudo bash

手动安装

喜欢自己折腾的看这里，手动装也不复杂：

  1. 下载最新安装包
1

wget https://www.geckosweb.cn/geckos/{版本号}/geckos.zip
  1. 解压到位
1
2

unzip geckos.zip -d /opt/geckos
chmod +x /opt/geckos/geckos
  1. 加个全局命令
1

ln -s /opt/geckos/geckos /usr/local/bin/geckos

  1. 配置文件已在程序目录，无需额外操作

  2. 注册 systemd 服务

cat > /etc/systemd/system/geckos.service << 'EOF'
[Unit]
Description=Geckos Web OS
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/geckos
ExecStart=/opt/geckos/geckos serve
Restart=always
RestartSec=5
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
EOF
