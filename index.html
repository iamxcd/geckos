功能特性

一站式服务器管理，告别繁琐的命令行操作

Web 终端

无需 SSH 客户端，浏览器直接执行 Linux 命令，随时随地远程管理服务器

文件管理

拖拽上传、在线编辑、压缩解压，Linux 服务器文件可视化管理，像操作本地电脑一样简单

Docker 管理

Docker 容器可视化管理，一键部署 Home Assistant、Jellyfin、qBittorrent 等应用

系统监控

实时查看 CPU、内存、磁盘使用情况，Linux 服务器状态一目了然，守护你的小主机

进程管理

查看 Linux 运行中的服务，一键结束异常进程，Web 端管理系统服务

插件扩展

一键安装 AdGuard Home、Alist、PhotoPrism 等个人服务器插件，扩展面板能力

AI 应用集成

一键部署 OpenClaw 龙虾AI、Hermes 爱马仕AI 等 AI 工具，养龙虾养马走起，轻松搭建私有 AI 助手，大模型本地化运行
